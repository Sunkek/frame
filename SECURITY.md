# Security policy

## Supported versions

| Version | Supported |
|---------|-----------|
| 0.x (latest) | ✅ |
| older minor versions | ❌ |

Once v1.0.0 is released, the latest minor release on each major version will
receive security fixes.

## Reporting a vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Report them privately via GitHub's built-in security advisory system:
**Security → Report a vulnerability** on the repository page.

Include:
- A description of the vulnerability and its potential impact
- Steps to reproduce or a minimal proof of concept
- The version of samsara affected

You can expect an acknowledgement within 72 hours and a resolution timeline
within 7 days for confirmed issues.
